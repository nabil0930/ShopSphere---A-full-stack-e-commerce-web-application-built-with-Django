# 🛒 ShopSphere — E-Commerce Platform

A full-stack e-commerce web application built with Django & Bootstrap 5.

## ✨ Features
- User Registration & Login with authentication
- Product listing with category filtering
- Search, Trending & Offers filters
- Add to Cart with quantity tracking
- Order placement with delivery address management
- Responsive premium UI with Bootstrap 5

## 🛠 Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, Bootstrap 5, FontAwesome
- **Database:** SQLite3
